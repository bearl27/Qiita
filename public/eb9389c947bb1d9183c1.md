# 原因
MetaQuestでは新しくアカウントを作成すると，開発者モードの設定をMetaのスマホアプリを使って再設定する必要があったようです。

# 解決方法
ホストアカウントのMetaアプリを開いて，再度開発者モードを設定する!

# おわりに
